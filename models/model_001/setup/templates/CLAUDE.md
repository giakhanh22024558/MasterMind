# <PROJECT NAME> — workspace onboarding

> Working folder cho BA work, scaffolded by **MasterMind model_001** `/set-up`.
> Đọc file này khi mở session mới.

## Folder layout (Core Rule)

```
<working-folder>/
├── MasterMind/        ← skill library (clone từ GitHub, read-only)
├── input/             ← raw materials (SRS docx, CR files, screenshots…)
├── context/           ← .md sidecars auto-generated by agent (đọc tiết kiệm token)
├── output/            ← deliverables (.docx, .xlsx, .drawio, .json…)
├── docs/              ← artifact templates user làm việc trực tiếp
└── conventions/       ← project overrides cho defaults skill
```

## Workflow chung

1. **Drop raw material** vào `input/` (file gì cũng được)
2. **Agent đọc + tạo `context/<file>.md`** sidecar (Core Rule Layer 1)
3. **User mở `docs/<artifact>.md`** rồi yêu cầu agent fill content
4. **Agent dùng skill model_001 phù hợp** — không tự invent format
5. **Deliverable cuối** ghi vào `output/` (vd `.docx`, `.xlsx`)

## Project conventions (override defaults)

Mở `conventions/<skill>-conventions.md` để override:
- ID formats (vd EPIC-XX vs E-XX vs Epic-XX)
- Dropdown values (Priority levels, Decision options, …)
- Color schemes
- Sheet/file naming

Bỏ trống = dùng default của skill.

|---|---|---|

## Common commands

```
"sync drive"           → pull latest từ Google Drive về context/ (nếu project setup Drive sync)
"create story under FEAT-XX"  → agent dùng google_sheets API hoặc edit docs/backlog.md
"gen Jira tasks"       → agent dùng skill integration/jira → output/jira/*.md
"build SRS"            → agent dùng skill document/srs → output/<project>-SRS.docx
"render ERD .drawio"   → agent render Mermaid → output/<project>-ERD.drawio
```

## Core Rule reminders

- ✅ `input/` = user-managed (drop file, đừng để agent ghi)
- ✅ `context/` = agent-managed (md sidecar, không edit thủ công — sẽ bị overwrite next sync)
- ✅ `output/` = deliverables — agent ghi, user xem/export
- ✅ Mọi file binary trong output/ phải có `.md` sidecar trong context/ (đọc tiết kiệm token)
- ❌ Đừng edit `MasterMind/` — đó là skill library, sync từ GitHub
